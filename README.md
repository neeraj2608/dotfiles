Dotfiles
----

###Installation:
<pre>
$> cd
$> wget https://bootstrap.pypa.io/get-pip.py
$> sudo python get-pip.py
$> rm get-pip.py
$> git clone git@github.com:neerajrao/dotfiles.git
$> cd dotfiles
$> chmod +x git-submodule-init.sh
$> ./git-submodule-init.sh
$> cp .vimrc ~/.
$> cp .bash_aliases ~/.
$> cp .gitconfig ~/.
$> cp -r .vim ~/.
</pre>
